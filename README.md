# Surveillance-during-extreme-weather-condition-
The Surveillance of Extreme Weather Conditions project is an intelligent weather monitoring and alert system designed to detect and report extreme weather events such as heavy rainfall, floods, heatwaves, cyclones, thunderstorms, and strong winds. The system collects real-time environmental data from sensors, processes the information, and generates alerts when weather parameters exceed predefined thresholds.
The project aims to provide early warnings and real-time monitoring to help reduce the impact of natural disasters and improve public safety.
 Objectives
Monitor weather conditions in real time.
Detect extreme weather events automatically.
Generate alerts and notifications for emergency situations.
Store historical weather data for analysis.
Support disaster management and decision-making processes.
 Features
Real-time weather data collection.
Temperature, humidity, rainfall, and wind speed monitoring.
Extreme weather detection based on threshold values.
Alert notifications via SMS, Email, or Mobile Application.
Interactive dashboard for data visualization.
Historical weather data storage and analysis.
User-friendly interface for monitoring weather conditions.

 Modules
1. Data Collection Module
Collects environmental data from weather sensors.
Monitors temperature, humidity, rainfall, and wind speed.
2. Data Processing Module
Cleans and processes collected data.
Converts raw sensor values into meaningful information.
3. Extreme Weather Detection Module
Compares sensor readings with predefined thresholds.
Identifies abnormal weather conditions.
4. Alert Generation Module
Sends warning notifications to users and authorities.
Supports Email, SMS, and mobile alerts.
5. Database Management Module
Stores weather records and alert history.
Maintains historical data for future analysis.
6. Visualization Dashboard Module
Displays real-time weather statistics.
Provides charts and reports for analysis.
